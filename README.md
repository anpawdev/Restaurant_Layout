# Restaurant_Layout

Simple restaurant website

## Technology:
- HTML5
- CSS3
- JavaScript
- Bootstrap

## Getting started:

Visit: https://annapawlik.github.io/Restaurant_Layout/

OR 

Clone repository
```
git clone git@github.com:AnnaPawlik/Restaurant_Layout.git
```

Install dependencies
```
npm install
```

Running development
```
// in main folder
npm run watch
```
